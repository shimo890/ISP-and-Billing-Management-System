# ISP Metrics Dashboard

## Project Overview

This project is a metrics dashboard focused on displaying core business metrics for ISP companies, including KPIs, revenue trends, and customer distribution.

## Architecture

### Frontend

- React + Vite for UI

### Backend

- Django + DRF + SQLite for API and data

## Core Business Flow

The application displays real-time dashboard metrics with date range filtering.

## Key Frontend Modules

- Dashboard page with charts and KPIs

## Technology Stack

- Backend: Django, Django REST Framework, SQLite
- Frontend: React, Tailwind CSS, Recharts

## Local Run Guide

### Backend

From `backend-api/`:

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver

### Frontend

From `frontend/`:

npm install

npm run dev
