# ISP Billing Dashboard - User Guide (Minimal)

## Overview

This guide explains the essential workflow for using the ISP Billing dashboard and core billing operations.

## Dashboard

The dashboard displays the key operational metrics:

- Total Revenue
- Active Customers
- Collection Rate (%)
- Outstanding Balance
- Weekly Revenue Trend
- Monthly Revenue Comparison

### How to use

1. Review KPI cards for current performance.
2. Check weekly and monthly trends for movement over time.
3. Use refresh to load the latest figures.

## Customers

### Create customer

1. Open `Customers`.
2. Click `Add New Customer`.
3. Enter basic details:
   - Name
   - Phone
   - Address
   - Customer Type
   - Status
4. Click `Save`.

### Manage customer

- Edit: open a customer and update details.
- Delete: remove a customer from the list.
- Search/filter: find by name, type, or status.

## Bill Entries

1. Open `Data Entry`.
2. Create a new bill for a selected customer.
3. Add service package details and billing period.
4. Save the bill.

## Invoices

1. Open `Invoices`.
2. Select customer and billing period.
3. Generate invoice preview.
4. Confirm and save.

## Payments

1. Open `Payments`.
2. Select customer and unpaid invoices.
3. Enter payment amount, method, date, and reference.
4. Save payment to update balances.

## Best Practices

- Keep customer records current.
- Enter bills and payments promptly.
- Review dashboard metrics daily.
- Verify amounts before saving transactions.

## Help

If an action fails, recheck required fields and retry. For access issues, contact your system administrator.
