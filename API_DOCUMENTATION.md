# API Documentation (ISP Metrics Dashboard)

Base URL (local):

- `http://127.0.0.1:8000/api`

---

## Dashboard Analytics (`/bills/analytics`)

- `GET /bills/analytics/dashboard/` - Core dashboard metrics (KPIs, revenue trends, customer distribution)

Query parameters:
- `start_date` - Start date for filtering (YYYY-MM-DD)
- `end_date` - End date for filtering (YYYY-MM-DD)
